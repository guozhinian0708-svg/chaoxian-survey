📋 发布指南

宁波市公路超限运输管理调查问卷 · 在线发布与统计

🚀 方案一：本地运行 + 内网穿透（推荐，5分钟完成）

数据存在你的电脑上，通过公网链接分享给各执法队填写，后台自动汇总。

第一步：启动本地服务

1
打开终端（Terminal），运行启动脚本：
cd /Users/guozhinian/WorkBuddy/2026-06-23-16-50-32/超限运输问卷 bash start.sh
2
等待公网链接生成，终端会显示类似：
✅ 公网链接已生成！ 📝 填表链接: https://xxx.loca.lt 📊 统计后台: https://xxx.loca.lt/admin

第二步：分享链接

3
将填表链接发给各执法队
各队打开链接 → 填写问卷 → 点击提交
⚠️ 对方首次打开链接时，会看到一个 "Click to Continue" 按钮页面，点击即可进入问卷。这是 localtunnel 的安全验证，只需操作一次。

第三步：查看统计

4
打开统计后台链接（/admin），实时查看：
• 总提交数、今日新增
• 是/否题柱状统计图
• 逐条详情查看
• CSV 导出（可用 Excel 打开）
• JSON 备份/还原
💡 如何停止：在终端按 Ctrl+C 即可。下次需要时重新运行 bash start.sh。
💡 链接有效期：只要你的电脑开机、终端不关，链接一直有效。每次重启电脑需要重新运行脚本。

☁️ 方案二：部署到免费云平台（长期方案）

如果你的问卷需要长期在线收集（不依赖电脑开关机），可以部署到 Render.com（免费额度足够使用）。

操作步骤概要：

1
注册 Render.com → 用 GitHub 账号登录 render.com
2
创建 Web Service → 选择 "Build and deploy from a Git repository"
3
配置参数：
• Build Command: npm install
• Start Command: node server.js
• 免费额度：每月 750 小时运行时间（足够）
4
获得永久链接，如 https://xxx.onrender.com
⚠️ 此方案需要先将项目上传到 GitHub，并在 Render 中关联。如需协助部署，对我说"帮我部署到 Render"即可。

📊 数据安全说明

  • 方案一：数据仅存储在 你的电脑 上的 data.json 文件中
  • 方案二：数据存储在 Render 服务器的临时文件系统中（建议定期导出备份）
  • 问卷不收集任何个人敏感信息，仅包含单位名称、联系人、电话和工作数据
